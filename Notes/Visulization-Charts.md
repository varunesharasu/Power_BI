\# All Charts Are 2D



\## Bar Family



\### Types of Charts



\* Column Chart

\* Bar Chart



\### Bar Family



There are two types of charts:



\*\*1. Column Chart\*\*



\* Vertical representation of bars.

\* Represents categorical analysis.

\* \*\*X-axis:\*\* Categories

\* \*\*Y-axis:\*\* Values



\*\*2. Bar Chart\*\*



\* Horizontal representation of bars.

\* Represents categorical analysis.

\* \*\*X-axis:\*\* Values

\* \*\*Y-axis:\*\* Categories



\## Multi-Category Analysis



\### 1. Clustered Column Chart



\* Vertical representation.

\* Multi-category analysis.

\* \*\*X-axis:\*\* Categories

\* \*\*Y-axis:\*\* Values

\* Bars are arranged side by side.



\### 2. Stacked Column Chart



\* Vertical representation.

\* Multi-category analysis.

\* \*\*X-axis:\*\* Categories

\* \*\*Y-axis:\*\* Values

\* Bars are stacked one above another.



\### 3. Clustered Bar Chart



\* Horizontal representation.

\* Multi-category analysis.

\* \*\*X-axis:\*\* Values

\* \*\*Y-axis:\*\* Categories

\* Bars are arranged side by side.



\### 4. Stacked Bar Chart



\* Horizontal representation.

\* Multi-category analysis.

\* \*\*X-axis:\*\* Values

\* \*\*Y-axis:\*\* Categories

\* Bars are stacked one after another.
